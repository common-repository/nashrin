Nashrin is a simple native advertising network for web and mobile, which supports only Persian/Farsi language now.

== Description ==

	This plugin enables WordPress users to easily integrate Nashrin's content recommendation widgets on their site. Generate direct revenue by surfacing Nashrin's high quality sponsored content recommendations.
	
	Notice: To use this plugin you first need to create widgets for your site in http://nashrin.com

== Installation ==

Installation Process

* Log into the WordPress admin for your site
* Go to 'Plugins' -> 'Add New'
* Search for Nashrin. Once found click on 'Install Now'
* Select 'Activate Plugin' from the download page or go to 'Plugins' -> 'Installed Plugins' -> 'Activate Plugin'
